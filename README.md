# Script to generate the RSI of a crypto symbol

The script is generated using [ChatGPT](https://openai.com/blog/chatgpt/)

## Usage

```
# in terminal run
deno run --alow-net run.ts -c BNBBUSD -i 1d -p 1000
```

## Parameters

```
-c cypto symbol
-i interval (1d, 1h 4h 1w)
-p period (max 1000)
```
